# FavForge — Favicon Generator

**FavForge** is a free, browser-based favicon generator that lets you craft pixel-perfect favicons in seconds. Choose from 300+ handmade emoji icons, upload your own image, apply solid or gradient backgrounds (linear & radial), pick shapes, and export in every format you need.

## ✨ Features

- 🎨 **300+ Emoji Icons** across 12 categories (Tech, Business, Creative, Nature, Food, Travel, Gaming, Animals & more)
- 🌈 **Gradient Backgrounds** — Solid, Linear, and Radial with full angle control
- 📐 **Shape Options** — Square, Rounded, Circle
- 🖼️ **Custom Image Upload** — Use your own logo or SVG
- 🎯 **AI Color Palettes** — Generate harmonious color combos
- ↩️ **Undo/Redo** with keyboard shortcuts (Ctrl+Z / Ctrl+Shift+Z)
- 🔀 **Randomize** — Instant random favicon generation
- 📦 **Multi-Format Export** — PNG (16–512px), ICO, Apple Touch Icon, Android Chrome icons
- 📋 **Full Pack Download** — ZIP with all sizes, manifest.json & HTML snippet
- 🖥️ **Live Preview** — See your favicon at multiple sizes and in a browser tab mockup

## 🛠️ Tech Stack

- **React** + **TypeScript**
- **Vite** for blazing-fast dev
- **Tailwind CSS** + **shadcn/ui**
- **HTML5 Canvas** for rendering
- **JSZip** for pack generation
